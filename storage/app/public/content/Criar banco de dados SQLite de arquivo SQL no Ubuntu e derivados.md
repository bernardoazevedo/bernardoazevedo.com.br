# Criar banco de dados SQLite de arquivo SQL no Ubuntu e derivados

## Modelo
```
sqlite3 [nome_do_banco_de_dados] < [arquivo_sql]
```

## Exemplo
```
sqlite3 [banco.db] < [create.sql]
```
